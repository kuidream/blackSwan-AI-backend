## 1. 世界真相：三层宇宙架构 (The Truth)
### 1.1 表世界：诸天主神空间 (The Surface)
+ **本质：** 我们所处的现实物理世界，实则是一个正在经历**“数据热寂（Data Heat Death）”**的高维空间。
+ **现状：** 由于维度屏障破碎，来自多元宇宙的**“信息乱流”**（即原始文本数据）不断侵蚀现实。
    - _这就是为什么周一你感觉像在修仙，周二像在赛博废土——这是现实世界的底层代码在被外部数据覆写。_
+ **玩家定位：****诸天行者 / 适格者**。
    - 只有佩戴了**“特异点终端”**（智能戒指/手表）的人，才能感知到这些侵蚀，并从中提取能量。普通人只觉得那天“运气不好”或“气氛压抑”。

### 1.2 里世界：TripX 模拟仓 (The Inner)
+ **本质：** 由 TripX 公司 (AERA) 在主神空间中心建立的**“绝对封闭算力黑盒”**。
+ **为何是方舟？**
    - 它是唯一能抵抗外部“信息乱流”的地方。这里没有修仙、没有魔法，只有**严酷、冰冷、逻辑严密的金融法则**。
    - **TripX 声称：** 只要按照 AERA 的规则运行，就能计算出逃离主神空间的**“奇点坐标”**（即船票）。
+ **危机：** 随着主神空间（外部）的崩塌加剧，TripX 的服务器算力逐渐枯竭，内部开始爆发**金融海啸（黑天鹅事件）**。它急需外部的高维能量来维持运转。

## 3. 源点经济学 (Source Economics)
**【源点 (Source)】** 是连接双重世界的唯一硬通货。它是“高维干涉力”的量化单位。

+ **源点 (Source) = 现金 (Cash/Capital)。** 这是账户里的流动资金。
+ **AERA = 股票资产 (Assets)。**
+ **关系：**
+ 你在**表世界（现实）打工（跑步/睡觉），赚取的是源点（工资）**。
+ 你在**里世界（TripX）**，用**源点**购买 **AERA**。
+ **AERA 下跌**——**浮亏**（账面资产缩水），**不会**直接扣除你的源点余额。
+ **平仓/卖出**—— 浮盈/浮亏变现，源点才会增加或减少。
+ **内幕消息/技能**—— 属于**增值服务**，需要直接消耗源点（现金）购买。

### 3.1 产出机制（表世界 -> 源点）
我们不再使用枯燥的“步数换金币”，而是使用**【行为共鸣率 (Resonance Rate)】**。

产出源点 = 基础行为值 X 风格共鸣系数 X 设备加成

+ **基础行为值：** 1000步 ≈ 10点；1小时深睡 ≈ 50点。
+ **风格共鸣系数：**
    - 在“武侠日（要求静心）”去“剧烈跑步”，共鸣系数 = **0.5**（事倍功半）。
    - 在“武侠日”进行“冥想/深呼吸”，共鸣系数 = **2.0**（事半功倍）。
+ **体验目标：** 玩家每天早上醒来看一眼风格，决定今天怎么“活”。

### 3.2 消耗机制（源点 -> 里世界）
源点在 TripX 里不是“钱”，而是**“修改现实的权限”**。

| **消耗行为** | **对应 TripX 效果** | **剧情解释 (Flavor Text)** |
| --- | --- | --- |
| **自动注入资金** | 账户增加源点本金 | “将高维算力转化为虚拟货币。” |
| **高维干涉** | **强行拉升 K 线 (反黑天鹅)** | **“本体通过购买内幕消息，知道K线会反转。”** |
| **心智覆写** | 解锁/重置女主对话 | “用高维信息流冲刷了她的记忆海马体。” |
| **San值锚定** | 清除负面情绪/Debuff | “来自本体的灵魂链接稳住了你的心神。” |


### 3.3 汇率与通胀 (Exchange Rate)
为了维持游戏寿命，源点对 AERA 资金的汇率随 **P0-P7 阶段** 动态贬值（通胀）。

+ **P0 阶段：** 1 源点 = 100 AERA 币。
+ **P7 阶段：** 1 源点 = 1 AERA 币（末日临近，算力贬值）。
+ **设计意图：** 逼迫玩家在后期必须在“现实中更拼命地运动”或者“在游戏里更高超地操作”。

## 4. 游戏单日循环 (The Game Day Loop)
基于上述设定，玩家的一天（游戏内一天）是这样度过的（类似于同校生）：

1. **08:00 - 觉醒（主神广播）：** AI 宣布今日加载《武侠》补丁。玩家决定今天“以静制动”（控制心率）。
2. **09:00~18:00 - 采集（表世界）：** 玩家在现实生活中保持平和。手环提示：“正在吐纳真气... 源点+50”。
3. **20:00 - 入仓（TripX）：** 玩家打开 APP，进入模拟仓。
4. **20:30 - 博弈（里世界）：** 遭遇金融风暴。玩家消耗白天攒下的“真气（源点）”发动【高维干涉】，强行止损。
5. **23:00 - 休眠（主神空间）：** 玩家睡觉，系统结算今日 San 值恢复情况。

## 5. TripX 的崩塌与终局 (The Collapse)
### 5.1 黑天鹅的真相
TripX 里的每一次**“黑天鹅事件”**（股市崩盘、病毒爆发），本质上是主神空间的**“信息乱流”击穿了 TripX 的防火墙**。

+ _例：当主神空间处于“克苏鲁风格”时，TripX 里不仅股票跌，K线图还会长出触手，女主会说出不可名状的疯话。_

### 5.2 唯一的船票
+ **定义：** 这不是一张物理船票，而是一串**“完美的不被侵蚀的代码序列”**。
+ **获取条件：**
    1. AERA 资产达到阈值（证明你有逻辑能力）。
    2. 源点注入总量达到阈值（证明你有高维能量支持）。
    3. 女主好感度/觉醒度达标（需要一个本地的“灵魂容器”）。
